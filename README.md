# 수박 자르기 게임
유니티 학원 강의를 통해 만든 게임


---
# 소개
## 게임 방법
+ 위에서 떨어지는 과일을 제어
+ 같은 과일은 합쳐짐.
+ 과일이 합쳐질때마다, 점수 획득
+ 과일은 7가지 존재
+ 노란색 경계선에서 3초간 과일이 존재 시, 게임 오버
